# SO_TC_02_Interprocess_Communication

## Execution Instructions

### Initialize Go Modules

```
go mod tidy
```

### Run the Simulator from the Project Root

```
go run main.go
```

### Run the Simulator as Shown in the Video (from `simulator` folder)

```
cd simulator
go mod tidy
go run ./cmd/simulator
```
